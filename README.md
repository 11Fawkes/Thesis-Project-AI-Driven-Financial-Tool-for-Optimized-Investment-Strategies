⚡ **AI-Powered Stock Analysis & Prediction – LSTM, SMA & SHAP**  

**Overview**

This project is an interactive AI-powered financial analysis dashboard built using Python, Streamlit, TensorFlow, and SHAP.
The goal of this system is not just to predict stock prices, but to:

Analyze trends using traditional financial indicators

Train a sequence-based deep learning model (LSTM)

Evaluate model performance using proper validation techniques

Explain model predictions using SHAP (Explainable AI)

**This project bridges:**

Financial intuition + Machine Learning + Interpretability

**🔍 What Problem Does This Solve?**

Most stock prediction projects online:

Ignore proper train/test splitting
Overfit historical data
Provide no explanation for predictions
Present black-box models without transparency

**This project focuses on:**

Generalization (train / validation / test discipline)
Transparent forecasting
Human-interpretable indicators
Model explainability via SHAP

It is built for educational, research, and decision-intelligence demonstration purposes.
