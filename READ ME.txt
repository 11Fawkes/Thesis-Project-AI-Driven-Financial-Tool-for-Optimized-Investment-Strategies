📈 AI-Driven Financial Analysis & Explainable Stock Forecasting Dashboard
🔎 Overview

This project integrates Machine Learning, Technical Analysis, and Explainable AI into an interactive Streamlit dashboard for stock analysis and forecasting.

It combines:

LSTM (Long Short-Term Memory) for stock price forecasting

SMA (Simple Moving Average) for trend detection

RSI & MACD for momentum and strength analysis

SHAP (SHapley Additive Explanations) for model interpretability

The goal is not just to predict stock prices, but to:

Detect market trends

Evaluate model performance responsibly

Prevent overfitting

Explain model decisions transparently

Demonstrate disciplined financial AI modeling

🎯 What Problem Does This Solve?

Many stock prediction projects:

Overfit historical data

Ignore train/test discipline

Use black-box models

Provide no explanation for predictions

Present unrealistic forecasting claims

This project focuses on:

✅ Proper time-based train / validation / test splitting
✅ Overfitting awareness via loss curves
✅ Transparent prediction evaluation
✅ Explainable AI using SHAP
✅ Integration of human financial intuition with ML

It is built as an educational and decision-intelligence demonstration tool, not a trading system.

The system is structured in four conceptual layers:

1️⃣ Data Layer

Retrieves stock data (Yahoo Finance with fallback)

2️⃣ Feature Engineering Layer

Generates financial indicators:

SMA (trend detection)

LSTM sequence model

Chronological 70/15/15 train/validation/test split

SHAP DeepExplainer (with fallback)

Global feature importance

Per-prediction explanation

Heatmap visualization

📊 Understanding the Financial Indicators

This section helps non-traders understand the signals.

📈 Simple Moving Average (SMA)

SMA smooths price data to identify the true direction of movement.

Example:
SMA(30) = average closing price of last 30 days
SMA(60) = average closing price of last 60 days

Why Use Two SMAs?

When short-term SMA crosses above long-term SMA:

→ Recent prices are stronger than historical average
→ Possible bullish trend

When it crosses below:

→ Possible bearish trend

This creates buy/sell signals.

SMA helps filter noise from daily price fluctuations.

📉 Relative Strength Index (RSI)

RSI measures momentum imbalance between buyers and sellers.

It compares:

Average recent gains

Average recent losses

Interpretation

RSI > 70 → Market may be overbought
RSI < 30 → Market may be oversold

Important:
RSI does NOT predict reversal.
It indicates market exhaustion or imbalance.

📊 MACD (Moving Average Convergence Divergence)

MACD measures trend strength.

It compares short-term and long-term exponential moving averages.

If MACD increases:
→ Trend gaining strength

If MACD decreases while price rises:
→ Trend weakening

MACD helps detect potential momentum shifts.

🤖 Machine Learning Model: LSTM
What is LSTM?

LSTM is a neural network specialized for sequence data.

It remembers patterns across time.

In this project:

The model observes the last N days of prices
and predicts the next day’s closing price.

⏳ Look-Back Window

Defines how many past days the model sees.

Example:
Look-back = 10
Model uses 10 past days to predict the 11th.

Small window → short-term focus
Large window → longer trend memory

📚 What is an Epoch?

One epoch = one full pass over the training dataset.

Too few epochs:
→ Model underfits (does not learn enough)

Too many epochs:
→ Model overfits (memorizes noise)

Early stopping monitors validation loss to stop training when overfitting begins.

📉 Train / Validation / Test Split

Time-series must be split chronologically.

70% → Training
15% → Validation
15% → Testing

Why?

To simulate real-world forecasting.

The model must predict data it has never seen.

This prevents data leakage.

📊 Evaluation Metrics
RMSE (Root Mean Squared Error)

Measures average prediction error.

Penalizes large mistakes heavily.

Lower RMSE = Better predictive accuracy.

MAE (Mean Absolute Error)

Measures average absolute difference between prediction and actual price.

More intuitive but less sensitive to large errors than RMSE.

🔍 Explainable AI (SHAP)

Most neural networks are black boxes.

SHAP explains:

Why the model predicted what it did.

Each past input day receives a SHAP value.

Positive SHAP:
Pushes prediction upward.

Negative SHAP:
Pushes prediction downward.

SHAP Visualizations
1️⃣ Global Importance Plot

Shows which past days influence predictions most overall.

2️⃣ Per-Instance Explanation

Explains one specific forecast.

3️⃣ SHAP Heatmap

Shows explanation stability across multiple predictions.
