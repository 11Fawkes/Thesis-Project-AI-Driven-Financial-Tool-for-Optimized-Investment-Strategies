📊 Financial Indicators Explained
🔹 Simple Moving Average (SMA)

SMA smooths price data to reveal trend direction.

SMA(30) → short-term trend

SMA(60) → long-term trend

When SMA(30) crosses above SMA(60):

→ Recent momentum stronger than historical average

This generates Buy/Sell crossover signals.

Purpose:
Helps separate real trends from market noise.

🔹 Relative Strength Index (RSI)

RSI measures recent buying vs selling strength.

RSI > 70 → market may be overbought

RSI < 30 → market may be oversold

Important:

RSI does not predict reversal.
It signals momentum imbalance.

🔹 MACD (Momentum Indicator)

MACD compares short-term and long-term exponential averages.

If MACD rises:

→ Trend gaining strength

If MACD falls while price rises:

→ Trend weakening

This helps detect potential momentum shifts.

🤖 Machine Learning Model (LSTM)
What is LSTM?

LSTM (Long Short-Term Memory) is a neural network designed for time-series data.

It learns patterns from sequences.

In this project:

The model observes the last N days of prices
and predicts the next day’s closing price.

⏳ Look-Back Window

Defines how many past days the model sees.

Example:

Look-back = 10
The model uses 10 previous days to predict day 11.

Smaller window → short-term behavior

Larger window → longer memory

📚 What is an Epoch?

One epoch = one complete pass over the training dataset.

Too few epochs → underfitting

Too many epochs → overfitting

Early stopping prevents memorization.

📉 Train / Validation / Test Discipline

Data is split chronologically:

70% → Training

15% → Validation

15% → Test

Why?

To simulate real-world forecasting.

The model must predict unseen future data.

This prevents data leakage and unrealistic accuracy.

📊 Model Evaluation
RMSE (Root Mean Squared Error)

Measures average prediction error.
Penalizes large mistakes more heavily.

MAE (Mean Absolute Error)

Measures average absolute prediction difference.

Lower values indicate better performance.

🔍 Explainable AI (SHAP)
What SHAP Actually Is (Conceptually)

SHAP comes from Shapley values in game theory.

Imagine:

Multiple players contribute to a final outcome.
You want to fairly assign credit to each player.

In your model:

Players = past input days (Day -1, Day -2, etc.)

Outcome = predicted price

SHAP answers:

“How much did each input contribute to this prediction?”

Important:

SHAP does NOT say:

This caused the price

This is true in the real world

It says:

The model relied on this input.

That is very different.

🧠 In This Project: What Are the Features?

Your LSTM sees:

Past N days of closing prices.

Example (look-back = 10):

Day -10

Day -9

Day -8

...

Day -1

The model uses those 10 values to predict the next day.

SHAP explains:

How much each of those 10 days influenced the output.

📊 SHAP Graph Interpretations
1️⃣ SHAP Summary Plot (Dot Plot)

This is the most important SHAP graph.

What it shows:

Each row = one feature (e.g., Day -1)

Each dot = one test prediction

X-axis = SHAP value (impact on prediction)

Right side → pushes prediction up
Left side → pushes prediction down

How to Interpret

If Day -1 has wide spread values:

→ The model heavily depends on yesterday’s price.

If Day -8 has very small spread:

→ The model barely uses that day.

If recent days dominate:

→ Model is momentum-focused.

If older days matter:

→ Model sees longer-term memory.

This reveals the type of strategy your model has implicitly learned.

2️⃣ Global Feature Importance (Bar Chart)

This averages absolute SHAP values.

It answers:

“On average, which inputs matter most?”

If ranking is:

Day -1 > Day -2 > Day -3 > …

That’s normal.

Markets are often most influenced by recent behavior.

If importance decays gradually:

→ Model learned proper temporal structure.

If importance appears random:

→ Model may not be stable.

3️⃣ Per-Instance Explanation

This explains one prediction.

Each bar shows:

How much that input pushed prediction up or down.

Green → pushes up

Red → pushes down

Example:

Prediction = 102

Day -1 = +0.8

Day -2 = +0.6

Day -3 = -0.2

This means:

Recent positive momentum influenced the forecast.

Professional interpretation example:

“The model’s upward prediction is primarily driven by strong recent gains rather than long-term trend.”

4️⃣ SHAP Heatmap

Rows = predictions

Columns = features

Color intensity = strength of influence

How to Interpret

If the same columns dominate across rows:

→ Model is consistent.

If patterns shift wildly:

→ Model may be unstable or overly sensitive to noise.

Consistency = trust.
Randomness = caution.

⚠️ What SHAP Does NOT Mean

SHAP does NOT say:

“This day caused price to rise.”

“This is a trading signal.”

“The market reacts this way.”

It says:

“The model relies on this input to compute its prediction.”

It explains model logic — not market causation.
